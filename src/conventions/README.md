# OpenPE Conventions

Domain-specific conventions grow automatically through the memory system.
See `memory/L1_domain/` for learned domain conventions.

Initial convention files will be created as analyses accumulate experience.

## Convention File Template

```yaml
domain: <domain_name>
learned_terminology:
  signal: "<domain-specific definition>"
  baseline: "<domain-specific definition>"
known_data_sources: []
known_pitfalls: []
effective_methods: []
```
